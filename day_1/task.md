# День1. Основы Vue и реактивности

Темы:
- создание Vue-приложения
- реактивные переменные
- computed-свойства
- интерполяция текста в шаблоне.

## Практическое задание

В этом уроке ты создашь свой рейтинг любимых книг.

Цель задания:
- научиться выводить списки в шаблоне vue
- освоить работу с реактивными переменными

## Порядок выполнения

1. Сделай форк учебного репозитория в github https://github.com/rsukhar/startup-hacker-course#
2. склонируй проект на локальную машину и перейди в папку day_1
2. создай в day_1 vue-проект, следуя инструкциям в официальной документации
3. создай компонент главной страницы приложения 
4. объяви реактивную переменную со списком книг (массив с 3-5 объктами)
5. объекты должны хранить следующую информацию: название, описание, жанр, обложка книги и рейтинг
6. выведи данные в template с помощью v-for (не забудь добавить атрибут key)
7. выведи рейтинг в виде звездочек (лучше использовать svg-иконки).
